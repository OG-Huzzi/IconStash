Create Pixel-Perfect SVG Icons

A detailed study of grid mathematics, snapping lines, anti-aliased borders, and optical weight compensation.

1. The Necessity of Grid Snapping

An icon rendering fuzzy inside a layout is usually due to misaligned fractional pixels (e.g. `x="12.4" y="5.7"`). When a path coordinate lies between grid lines, the browser attempts to blend the pixel colors (anti-aliasing) to create the curve, resulting in blurry borders.

To avoid this, always set coordinates to round integer coordinates. Modern tools have a "snap to grid" configuration which handles this automatically during export.

2. Inner, Outer, and Centered Strokes

SVG strokes are centered on the vector path by default. If you have a 2px stroke on a line at `x="10"`, the stroke will extend 1px to the left (9px) and 1px to the right (11px).

This path layout causes rendering blurring if you use an odd-numbered stroke width (like 1px) on integer grid coordinates, because the stroke will split into halves (e.g. 0.5px). In these situations, offset path alignments by 0.5px (e.g. `x="10.5"`) to snap the stroke edges back onto integer boundaries.

3. Compensating for Optical Geometry

Circles, squares, and triangles possess different densities and optical weight. Standardizing them to the exact same pixel bounding box makes circles look too small and squares look too large.

To keep sizes visually balanced across a set, establish margins inside your grid. Squares should stay within a smaller internal boundary (e.g. 18px), while circles should stretch closer to the active border (e.g. 20px).

4. Testing on High-DPI Displays

Always review icon scales at their natural, target pixel size (typically 16px, 20px, or 24px) rather than zoomed-in on the designer's canvas. An icon that looks detailed at 800% zoom may collapse into a dense blob of lines on low-resolution displays.

Key Takeaways

  • Snap coordinates: Ensure coordinates end on whole numbers to keep lines crisp.
  • Odd widths: Offset 1px strokes by 0.5px to align with pixel grids.
  • Optical weights: Scale shapes visually so circles, squares, and triangles feel balanced.
